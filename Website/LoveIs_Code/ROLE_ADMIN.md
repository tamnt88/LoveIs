# Admin Roles & Features

## System
- Admin menus filtered by MenuGroup = "Admin".
- System config: email, footer, banners.

## Catalog
- Manage 3-level categories.
- Manage filters/variants.
- Manage products & images.

## Users
- Manage users/roles/permissions.
- Manage buyers/sellers.

## Orders
- Manage main and shop orders.
- Change order statuses (affects Sold30d).

## Community
- Optional moderation (if enabled).
