# Seller Roles & Features

## Account
- Seller portal: `/seller`.
- One shop per seller (1-1).

## Products
- Create/manage products for own shop.
- Manage variants, images, pricing.

## Orders
- Receive split shop orders (cf_shop_order).
- Update order statuses.
- Returns/refunds (mocked).

## Reviews
- View product & shop reviews.

## Finance
- Fees stored per order (platform, infra, payment, shipping).
- Reconciliation UI not built yet.

## Community
- Seller can post/chat like buyers.
- Posts show seller badge.
