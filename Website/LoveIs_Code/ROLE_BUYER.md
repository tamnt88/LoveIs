# Buyer Roles & Features

## Account
- Registration & login required to place orders.
- Email verification required before activation.
- OAuth Google/Facebook login.

## Shopping
- Browse categories & filters.
- Product detail: images, variants, price, shop.
- ViewCount increment on product detail view.

## Cart
- Add variants to cart.
- Cart grouped by shop.
- Bulk remove selected items.
- Limits based on total spent tiers.

## Checkout
- Address book selection.
- Shipping/payment method selection.
- Auto-save first address.

## Profile
- Profile: update display name, birthday.
- Bank: save payout/bank info.
- Addresses: manage & set default.
- Change password.
- Orders list + order detail timeline.

## Community
- Post, like, comment.
- Chat 1-1 or group.
- Header notifications.
